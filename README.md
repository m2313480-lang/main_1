 🥾 ShoeMaster Pro

[![Codacy Badge](https://app.codacy.com/project/badge/Grade/32ca7762fd4f41a6a8039ddc5b4cfb36)](https://app.codacy.com/gh/m2313480-lang/main_1/dashboard?utm_source=gh&utm_medium=referral&utm_content=&utm_campaign=Badge_grade)

ShoeMaster Pro — комплексное веб-решение для автоматизации работы обувного магазина. Система предоставляет гибкие возможности управления товарным ассортиментом, заказами и пользователями с разграничением доступа по ролям.

---

🛠️ Технологический стек

| Компонент | Технология |
|-----------|------------|
| Клиентская часть | React 18 + Vite |
| Серверная часть | FastAPI + Uvicorn |
| Хранение данных | Импорт из `data/products.xlsx` |
| Медиафайлы | Статическая раздача через `/media/*` |
| Аналитика кода | ESLint + Flake8 + Codiga |

---

Реализованный функционал

Система авторизации
- Ролевая модель: **администратор**, **менеджер**, **клиент**, **гость**
- Разграничение доступа к разделам системы

 Управление товарами
- Отображение товаров с фото, ценой, скидкой и остатком
- Интерактивная таблица с визуальными индикаторами:
  - Голубой фон — товар отсутствует на складе
  - Зеленый фон (#2E8B57) — скидка превышает 15%
  - Зачеркнутая цена + красный цвет — действующая скидка
- Поискпо всем текстовым полям (наименование, описание, производитель)
- Фильтрация по поставщику с возможностью сброса
- Сортировка по количеству на складе (возрастание/убывание) — для менеджера и администратора
- CRUD операции с товарами (только администратор)

Управление заказами
- Отдельный модуль для менеджера и администратора
- Полный CRUD заказов для администратора

 API
- RESTful эндпоинты для авторизации, товаров и заказов
- Автоматическая документация Swagger UI

 Быстрый старт

 1. Клонирование репозитория
```bash
git clone https://github.com/username/shoe-master-pro.git
cd shoe-master-pro
```

 2. Настройка бэкенда
```bash
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --reload --port 8000
```

 3. Настройка фронтенда (в новом терминале)
```bash
cd frontend
npm install
npm run dev
```

 4. Доступ к приложению
- **Веб-интерфейс:** http://localhost:5173
- **API документация:** http://localhost:8000/docs
- **ReDoc:** http://localhost:8000/redoc

API Эндпоинты

| Метод | Эндпоинт | Описание | Доступ |
|-------|----------|----------|--------|
| POST | `/auth/login` | Авторизация | Все |
| GET | `/products` | Получить товары | Все |
| POST | `/products` | Создать товар | Админ |
| PUT | `/products/{id}` | Обновить товар | Админ |
| DELETE | `/products/{id}` | Удалить товар | Админ |
| GET | `/orders` | Получить заказы | Менеджер+, Админ |
| POST | `/orders` | Создать заказ | Админ |
| PUT | `/orders/{id}` | Обновить заказ | Админ |
| DELETE | `/orders/{id}` | Удалить заказ | Админ |
